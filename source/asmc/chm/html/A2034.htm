Asmc Macro Assembler Reference

ASMC Nonfatal Error A2034

must be in segment block

One of the following was found outside of a segment block:

  • An instruction
  • A label definition
  • A THIS operator
  • A $ operator
  • A procedure definition
  • An ALIGN directive
  • An ORG directive
* ML 6.0 compatible error
