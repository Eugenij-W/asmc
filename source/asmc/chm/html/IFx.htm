Size of return code

.IF foo() == -1	 ; ax/eax/rax
.IFB foo() == -1 ; al
.IFW foo() == -1 ; ax
.IFD foo() == -1 ; eax

Signed compare

.IFS foo() > 0	; if first arg is reg then signed compare
.IFSB foo() > 0 ; 16/32/64-bit
.IFSW foo() > 0 ; 32/64-bit
.IFSD foo() > 0 ; 64-bit
