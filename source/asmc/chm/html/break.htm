.BREAK

.BREAK[1|2|3] [[.IF condition]]

Generates code to terminate a .WHILE or .REPEAT block if condition is true.

BREAK[1|2|3] is optional nesting level to terminate.

	.while 1
	    .break		; break .while 1
	    .while 2
		.break1		; break .while 1
		.while 3
		    .break2	; break .while 1
		    .while 4
			.break3 ; break .while 1
			.break2 ; break .while 2
			.break1 ; break .while 3
		    .endw
		.endw
	    .endw
	.endw
