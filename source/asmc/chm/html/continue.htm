.CONTINUE

.CONTINUE[1|2|3] [[.IF condition]]
.CONTL[0|1|2|3] [[.IF condition]]
.CONTL0[1|2|3] [[.IF condition]]

Generates code to jump to the top of a .WHILE or .REPEAT block if condition is true.

.CONTL[1|2|3] is optional nesting level to continue.

	.while 1
	    .continue		; continue .while 1
	    .while 2
		.contl1		; continue .while 1
		.while 3
		    .contl2	; continue .while 1
		    .while 4
			.contl3 ; continue .while 1
			.contl2 ; continue .while 2
			.contl1 ; continue .while 3
		    .endw
		.endw
	    .endw
	.endw

.CONTL0[1|2|3] jump's directly to START label: no TEST.

	.while 1
	    .contl0		; Jump to START label
	    .continue		; Jump to START label
	.endw
	.while eax
	    .contl0		; Jump to START label
	    .continue		; Jump to TEST label
	.endw
	.repeat
	    .contl0		; Jump to START label
	    .continue		; Jump to EXIT label
	    .break		; Jump to EXIT label
	.until 1
