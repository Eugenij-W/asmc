Directives Reference

OPTION optionlist

Enables and disables features of the assembler. Added options by ASMC include:

  • OPTION ASMC
  • OPTION CSTACK
  • OPTION SWITCH
  • OPTION LOOPALIGN
  • OPTION CASEALIGN
  • OPTION WSTRING

Conditional Control Flow

  • .SWITCH
  • .CASE
  • .ENDC
  • .DEFAULT
  • .ENDSW
  • .IFx
  • .ASSERT
