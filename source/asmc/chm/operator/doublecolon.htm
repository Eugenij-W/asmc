OPERATOR ::

edx::eax
label::
type::id proto ...
type::id proc ...

The first operator is used in INVOKE to create a 64-bit argument in 32-bit.

The second operator is used as a global label within a PROC or as a PUBLIC label outside.

The next two operators is used to define a class member with a hidden (_this) argument of type.
